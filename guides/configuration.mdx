Reze AI uses environment variables for all configuration. This guide covers all available settings and how to configure them.

## Setup

<Steps>
  <Step title="Copy environment template">
    Copy the example environment file to create your configuration:
    
    ```bash
    cp env.example .env
    ```
  </Step>
  
  <Step title="Configure required variables">
    At minimum, you must configure:
    - GLM API key for AI functionality
    - Resend API key for email sending
    - Database URL for persistence
  </Step>
  
  <Step title="Start the application">
    The application automatically loads settings from `.env` on startup.
    
    ```bash
    python -m src.api.app
    ```
  </Step>
</Steps>

<Warning>
  Never commit your `.env` file to version control. It contains sensitive API keys and credentials.
</Warning>

## Configuration reference

### Application settings

General application configuration for the FastAPI server.

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `DEBUG` | boolean | `true` | Enable debug mode for development |
| `HOST` | string | `0.0.0.0` | Server host address |
| `PORT` | integer | `8000` | Server port number |
| `LOG_LEVEL` | string | `INFO` | Logging level (DEBUG, INFO, WARNING, ERROR, CRITICAL) |
| `SENTRY_DSN` | string | - | Sentry DSN for error tracking (optional) |

<CodeGroup>
```bash Development
DEBUG=true
HOST=0.0.0.0
PORT=8000
LOG_LEVEL=DEBUG
```

```bash Production
DEBUG=false
HOST=0.0.0.0
PORT=8000
LOG_LEVEL=INFO
SENTRY_DSN=https://your-sentry-dsn@sentry.io/project
```
</CodeGroup>

### GLM 4.7 AI settings

Configuration for the GLM 4.7 language model from z.ai.

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `GLM_API_KEY` | string | **required** | GLM 4.7 API key from z.ai |
| `GLM_BASE_URL` | string | `https://open.bigmodel.cn/api/paas/v4` | GLM 4.7 base URL (OpenAI-compatible) |
| `GLM_MODEL` | string | `glm-4.7` | GLM model name |

<Note>
  Get your GLM API key from [https://open.bigmodel.cn/](https://open.bigmodel.cn/)
</Note>

```bash .env
GLM_API_KEY=your_glm_api_key_here
GLM_BASE_URL=https://open.bigmodel.cn/api/paas/v4
GLM_MODEL=glm-4.7
```

### Resend.com settings

Configuration for the Resend email API.

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `RESEND_API_KEY` | string | **required** | Resend API key |
| `RESEND_FROM_EMAIL` | string (email) | **required** | Default sender email address |
| `RESEND_BASE_URL` | string | `https://api.resend.com` | Resend API base URL |

<Note>
  - Get your Resend API key from [https://resend.com/api-keys](https://resend.com/api-keys)
  - The sender email must be from a verified domain in your Resend account
</Note>

```bash .env
RESEND_API_KEY=re_xxxxxxxxxxxx
RESEND_FROM_EMAIL=noreply@yourdomain.com
RESEND_BASE_URL=https://api.resend.com
```

### Memvid RAG settings

Configuration for the Memvid knowledge base and RAG functionality.

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `MEMVID_FILE_PATH` | string | `./memory.mv2` | Path to Memvid knowledge base file |
| `MEMVID_INDEX_KIND` | string | `basic` | Memvid index kind (`basic` or `advanced`) |

<Note>
  The Memvid file will be created automatically on first run. See [Populating Memvid](/guides/populating-memvid) for setup instructions.
</Note>

```bash .env
MEMVID_FILE_PATH=./memory.mv2
MEMVID_INDEX_KIND=basic
```

### Database settings

Configuration for the conversation history and email logs database.

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `DATABASE_URL` | string | `sqlite+aiosqlite:///./database` | SQLAlchemy database URL |
| `DATABASE_ECHO` | boolean | `false` | Enable SQLAlchemy query logging |

<CodeGroup>
```bash SQLite (Development)
DATABASE_URL=sqlite+aiosqlite:///./database
DATABASE_ECHO=false
```

```bash PostgreSQL (Production)
DATABASE_URL=postgresql+asyncpg://user:password@db-host:5432/reze
DATABASE_ECHO=false
```
</CodeGroup>

## Settings class

All configuration is managed through the `Settings` class in `src/config.py`, which uses Pydantic for validation and type safety.

```python src/config.py
class Settings(BaseSettings):
    """Main settings container with flat structure for .env reading."""

    # Application Settings
    debug: bool = Field(default=True, env="DEBUG")
    host: str = Field(default="0.0.0.0", env="HOST")
    port: int = Field(default=8000, env="PORT")
    log_level: str = Field(default="INFO", env="LOG_LEVEL")
    sentry_dsn: str = Field(default="", env="SENTRY_DSN")

    # GLM 4.7 AI Provider Settings
    glm_api_key: str = Field(default="test_glm_key", env="GLM_API_KEY")
    glm_base_url: str = Field(
        default="https://open.bigmodel.cn/api/paas/v4",
        env="GLM_BASE_URL"
    )
    glm_model: str = Field(default="glm-4.7", env="GLM_MODEL")

    # Resend.com API Settings
    resend_api_key: str = Field(default="re_test_key", env="RESEND_API_KEY")
    resend_from_email: EmailStr = Field(
        default="test@example.com",
        env="RESEND_FROM_EMAIL"
    )
    resend_base_url: str = Field(
        default="https://api.resend.com",
        env="RESEND_BASE_URL"
    )

    # Memvid RAG Settings
    memvid_file_path: str = Field(
        default="./memory.mv2",
        env="MEMVID_FILE_PATH"
    )
    memvid_index_kind: Literal["basic", "advanced"] = Field(
        default="basic",
        env="MEMVID_INDEX_KIND"
    )

    # Database Settings
    database_url: str = Field(
        default="sqlite+aiosqlite:///./database",
        env="DATABASE_URL"
    )
    database_echo: bool = Field(
        default=False,
        env="DATABASE_ECHO"
    )
```

### Validation

The Settings class includes automatic validation:

- **URL validation**: Base URLs must start with `http://` or `https://`
- **Email validation**: Email addresses are validated using Pydantic's `EmailStr`
- **Type validation**: All fields are type-checked at runtime

```python src/config.py
@field_validator("glm_base_url", "resend_base_url")
@classmethod
def validate_base_url(cls, v: str) -> str:
    """Validate base URL format."""
    if not v.startswith(("http://", "https://")):
        raise ValueError("Base URL must start with http:// or https://")
    return v
```

## Accessing settings

Settings are available globally through the `settings` instance:

```python
from src.config import settings

# Access any setting
api_key = settings.glm_api_key
from_email = settings.resend_from_email
debug_mode = settings.debug
```

Or use the `get_settings()` function for dependency injection:

```python
from src.config import get_settings

def my_function():
    settings = get_settings()
    return settings.port
```
